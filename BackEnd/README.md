## Getting Started:
> Open **3** Terminals and run each of the following command

_**Server:**_

```
cd BackEnd
npm install
npm start 
```

_**DataBase:**_

```
mongod
```

_**Angular:**_

```
cd FrontEnd
npm install
ng serve
```
